# My Personal Study Notes and Resources

A simple static website for organizing study notes and resources.

## Project Description

This project is a static HTML/CSS website that organizes course notes, resources, and study materials. It includes multiple pages for different subjects.

## How to Run

1. Clone the repository:
   ```
   git clone <your-repo-url>
   ```
2. Open `index.html` in any web browser.
3. No server or installation needed — it's a plain static site.

## Project Structure

```
/
├── index.html       - Homepage
├── notes.html       - Study notes page
├── about.html       - About page
├── style.css        - Main stylesheet
├── README.md        - Project documentation
├── CHANGELOG.md     - Version history
└── .gitignore       - Ignored files list
```
